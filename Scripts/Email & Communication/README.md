# Email & Communication

Scripts and references for sending emails and SMS via email gateways.

## Scripts

### Redacted-Send-Email-With-Attachement.py
Python script that sends an email with a file attachment via Gmail SMTP (SSL). Uses app passwords for authentication.

### Phone_Provider_Email_to_Text.txt
Reference list of email-to-SMS gateway addresses for major US carriers (AT&T, Verizon, T-Mobile, Sprint, etc.). Send an email to these addresses to deliver an SMS to a phone number.
